## Projetos em Destaque

### Sistema Oficina
Sistema completo de gerenciamento para oficinas mecânicas com arquitetura organizada e estrutura escalável.

**Principais recursos**
- CRUD completo de clientes, veículos e ordens de serviço
- Estrutura em camadas (Controller → Service → Repository)
- Integração com banco relacional
- API estruturada com validações e tratamento de erros

**Stack**
Node.js • Express • MySQL • REST API • Arquitetura modular

---

### HotelCompare
Plataforma de comparação de hotéis com foco em performance, integração e organização de dados.

**Principais recursos**
- Comparação estruturada de resultados
- Consumo de APIs externas
- Tratamento e normalização de dados
- Interface otimizada para leitura rápida

**Stack**
Node.js • APIs externas • Processamento de dados • Backend estruturado
